🗄️ Supabase Setup Guide

Connect a real shared database — data syncs in real time across all users.

Step 1 — Create a free project

Go to supabase.com → New Project → choose name & region → wait ~1 min.

Step 2 — Run SQL

Project → SQL Editor → New query → paste & run:

create table donors (
  id text primary key,
  fname text, lname text, roll text unique, email text unique,
  phone text, year text, dept text, blood text,
  hostel text default '', wa text default '',
  status text default 'available', date text,
  donations int default 0, verified boolean default false,
  created_at timestamptz default now()
);
create table alerts (
  id text primary key, patient text, blood text, units text,
  loc text, contact text, notes text default '', date text,
  matching int default 0, created_at timestamptz default now()
);
create table requests (
  id text primary key, name text, phone text, blood text,
  urgency text, date text, created_at timestamptz default now()
);
create table donations (
  id text primary key, did text references donors(id) on delete cascade,
  blood text, date text, loc text, note text default '',
  created_at timestamptz default now()
);
alter publication supabase_realtime add table donors, alerts, requests, donations;
alter table donors enable row level security;
alter table alerts enable row level security;
alter table requests enable row level security;
alter table donations enable row level security;
create policy "Public all" on donors for all using (true) with check (true);
create policy "Public all" on alerts for all using (true) with check (true);
create policy "Public all" on requests for all using (true) with check (true);
create policy "Public all" on donations for all using (true) with check (true);
Step 3 — Get credentials

Settings → API → copy Project URL and anon/publishable key.

Step 4 — Paste into main app file

Open the main HTML file and replace:

const SUPABASE_URL = 'https://yourproject.supabase.co';
const SUPABASE_ANON_KEY = 'your-anon-key-here';

✅ Done! All data now syncs in real time across every user.
