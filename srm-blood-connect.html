🗄️ Supabase Setup Guide

Follow these steps once to connect a real shared database. Data will instantly sync across all users in real time.

Step 1 — Create a free Supabase project

Go to supabase.com → New Project → choose a name & region → wait ~1 min for setup.

Step 2 — Run this SQL in the SQL Editor

Project → SQL Editor → New query → paste this → Run:

create table donors (
  id text primary key,
  fname text, lname text, roll text unique, email text unique,
  phone text, year text, dept text, blood text,
  hostel text default '', wa text default '',
  status text default 'available',
  date text, donations int default 0,
  verified boolean default false,
  created_at timestamptz default now()
);
create table alerts (
  id text primary key,
  patient text, blood text, units text, loc text,
  contact text, notes text default '', date text,
  matching int default 0,
  created_at timestamptz default now()
);
create table requests (
  id text primary key,
  name text, phone text, blood text,
  urgency text, date text,
  created_at timestamptz default now()
);
create table donations (
  id text primary key,
  did text references donors(id) on delete cascade,
  blood text, date text, loc text, note text default '',
  created_at timestamptz default now()
);
-- Enable realtime
alter publication supabase_realtime add table donors;
alter publication supabase_realtime add table alerts;
alter publication supabase_realtime add table requests;
alter publication supabase_realtime add table donations;
-- Allow public access (anon key)
alter table donors enable row level security;
alter table alerts enable row level security;
alter table requests enable row level security;
alter table donations enable row level security;
create policy "Public read" on donors for select using (true);
create policy "Public insert" on donors for insert with check (true);
create policy "Public update" on donors for update using (true);
create policy "Public delete" on donors for delete using (true);
create policy "Public read" on alerts for select using (true);
create policy "Public insert" on alerts for insert with check (true);
create policy "Public delete" on alerts for delete using (true);
create policy "Public read" on requests for select using (true);
create policy "Public insert" on requests for insert with check (true);
create policy "Public read" on donations for select using (true);
create policy "Public insert" on donations for insert with check (true);
Step 3 — Get your credentials

Project Settings → API → copy Project URL and anon public key.

Step 4 — Paste into this file

Open the HTML file in a text editor and replace these two lines near the top of the <script>:

const SUPABASE_URL = 'https://yourproject.supabase.co';
const SUPABASE_ANON_KEY = 'eyJhbGci...(your anon key)';

✅ That's it! Refresh the page — all data is now shared across users in real time.
